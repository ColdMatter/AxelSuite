Motors A/B

Two identical groups A and B represent two motors moving (tilting) the platform.

First button (Initialize) will move the motor to home position (hardware fixed) and then assign 0 to this position. That way the system can have reproducible initial position position.

Next button (Move [mm]) and the numeric field after are for move the respective motor to some position. The idea here is to motors to a horizontal position of the platform. The horizontality is measured by crossed level-meter. There are different types of such level-meters, but even bubble one will do the job. These two positions (for horizontal state) will be saved and after restart retrieved.

Status button will print in the log the status of the respective motor taken from the controller.

The last button ([mm]) will execute going home and move to retrieved position in one click. When the system starts pressing both (in sequence) buttons will recover the horizontal position.
 

Tilt

First button (Initialize) will execute initialization for motors A/B. Next button (Move to preset...) will tilt the platform to  the retrieved positions (A/B) and then will assign 0 tilt to this tilt position.

Move [mm] button will move the platform to required tilt measured in vertical displacement. Second (Move [rad]) will move the platform to required tilt measured in radians.

Stop button is for emergency stop, if some dangerous actions are in progress.  
 

Options

Speed [mm/s] allows to define the movement speed. Usually following the manufacturer recommendations.

Blacklash compensation is integrated procedure to compensate the backlash making any approach to a position only from one side. That means if the default approach is up and the user moves down the controller will move down pass the position with some predefine distance and then move up in order to approach the position from up. 

MEMS correction will switch On/Off the offset dodging parameter which allows to make a final adjustment of the horizontality using the electric signals for the MEMS.
